# Instrucciones rápidas para iniciar el backend

## Si PowerShell no deja ejecutar npm

Usa CMD en su lugar:

1. Abre CMD (no PowerShell)
2. Navega a la carpeta backend:
   ```
   cd C:\Users\fdhm3\OneDrive\Desktop\Fullstack-2\backend
   ```
3. Instala las dependencias:
   ```
   npm install
   ```
4. Inicia el servidor:
   ```
   npm start
   ```

## Verificar que el archivo de credenciales está en su lugar

Asegúrate de que `serviceAccountKey.json` está en la carpeta `backend/`.

## Acceder a Swagger

Una vez el servidor esté corriendo, abre en tu navegador:
```
http://localhost:5000/api-docs
```

O usa los links que agregué en el panel de administración (sidebar y dashboard).
